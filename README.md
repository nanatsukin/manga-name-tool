# Manga Studio - ネーム製作ツール

漫画制作のワークフロー（プロット → コンテ → ネーム）をブラウザ上で一貫して行えるWebアプリケーションです。

## 機能

### 3つの制作モード

- **プロットモード** - セリフ・ト書き・注意書きをページ単位で入力・管理。ドラッグ&ドロップで並び替え、ページ分割・結合に対応
- **コンテモード** - 各ページにコマ（カット）を作成し、手描きで絵コンテを描画。セリフをコマにドラッグ&ドロップで割り当て
- **ネームモード** - 漫画原稿のレイアウトを作成。コマとセリフを自由配置し、仕上がり枠・裁ち落とし・トンボのガイド表示付き。自動整列機能あり

### 書き出し・入出力

- **PNG書き出し** - 連番画像をZIPで一括ダウンロード
- **PSD書き出し** - レイヤー構造を保持したPSDファイルをZIPで出力（CLIP STUDIO等で編集可能）
- **テキスト出力** - ページごとのセリフをクリップボードにコピー
- **プロジェクト保存/読込** - JSON形式でローカルファイルに保存。IndexedDBによる自動保存にも対応

### その他

- ページ設定（キャンバスサイズ、仕上がり枠、裁ち落とし、基本枠マージン）
- フォント設定（フォントファミリー、基本フォントサイズ）
- 縦書きテキスト表示
- Undo/Redo対応
- モバイルレスポンシブ対応
- キーボードショートカット

## 技術構成

ビルドツール不要のシンプルな構成です。`index.html` をブラウザで開くだけで動作します。

- **Vue 3** (CDN) - リアクティブUI
- **Pinia** (CDN) - 状態管理
- **Tailwind CSS** (CDN) - スタイリング
- **html-to-image** - ページのPNG書き出し
- **ag-psd** - PSD形式書き出し
- **JSZip / FileSaver.js** - ZIP圧縮・ファイルダウンロード
- **idb-keyval** - IndexedDBによる自動保存

## ファイル構成

```
src/
  index.html        … メインHTML（テンプレート）
  style.css         … カスタムスタイル
  script.js         … オーケストレーター（モジュール結合・ウォッチャー）
  js/
    stores/          … Pinia ストア（状態管理を4ドメインに分割）
    core/            … 基盤モジュール（ユーティリティ）
    mode/
      plot/          … プロットモード（セリフD&D）
      conte/         … コンテモード（描画キャンバス・コマD&D）
      name/          … ネームモード（コマ・セリフ配置）
    ops/             … 横断的操作（ページ操作・キーボード・保存/読込・書出し）
tests/               … テストコード（Vitest）
```

## 使い方

1. `src/index.html` をブラウザで開く
2. **プロット**モードでセリフ・ト書きを入力
3. **コンテ**モードでコマを作成し、絵コンテを描いてセリフを割り当て
4. **ネーム**モードでコマとセリフを原稿上に配置
5. 必要に応じてPNG/PSD形式で書き出し

## 動作環境

モダンブラウザ（Chrome, Edge, Firefox, Safari）で動作します。
File System Access API に対応したブラウザ（Chrome, Edge）では上書き保存が利用できます。
